04 / 09
HARDWARE

硬件方案 — 霍尔传感器

01
线性霍尔传感器
非接触式检测
磁场强度变化
AH3505 / A1302
无磨损 · 精度高 · 寿命长
02
钕磁铁
贴附于手把背面
跟随把手旋转
薄片型安装
隐藏安装 · 无需改造手把
03
Arduino Pro Micro
USB HID 模拟游戏手柄
ADC 读取模拟电压
线性映射到游戏输入
即插即用 · 无需驱动
04
3D打印支架
固定传感器位置
精确安装角度
可定制外壳
支持品牌定制外壳
工作原理
手把旋转 → 磁铁靠近/远离霍尔传感器 → 磁场强度变化 → 输出电压变化 → ADC 读取 → 映射为游戏输入值
游戏玩法
05 / 09
GAMEPLAY

刹车测试挑战

01
反应刹车
红灯亮起 → 绿灯亮 → 拉手把刹车 → 测反应时间
测试指标
从绿灯亮起到手把开始拉动的时间
适合场景
反应速度测试 / 新手体验
02
刹车距离挑战
摩托车动画冲刺 → 看到障碍 → 拉手把刹车 → 看谁刹车距离最短
测试指标
从刹车到完全停下的距离（拉力越深刹车越急）
适合场景
性能展示 / 拉力手感体验
线性控制优势
手把拉力深浅实时映射为刹车力度，拉得越深刹车越急。
这让游戏体验与真实手感完全一致，真正感受产品性能。
支持
双人对战
