# 视觉风格基准

`Resources/template.png` 是本项目从 2026-06-15 起的强制视觉参考。后续每一步、每一个版本都必须优先贴近这张图的画面语言。

## 必须保持的风格特征

- 俯视赛道视角，镜头有轻微倾斜和真实转播感。
- 赛道为灰黑沥青，带轮胎痕、白线、红白路肩、绿色缓冲区和草地。
- UI 采用 MotoGP 转播式 HUD：黑色半透明信息块、白色大数字、红色强调色、紧凑排名列表。
- 右上角显示大计时，右侧显示分段数据，右下角显示圆形速度/力度仪表。
- 车辆标签贴近转播字幕条，而不是普通游戏卡片。
- 整体质感偏真实电视转播，可以有卡通化简化，但不能变成低龄、糖果色、扁平网页风。

## 禁止偏离

- 不使用大面积渐变背景、装饰性卡片或营销页面布局。
- 不使用圆润可爱、儿童游戏式按钮作为主视觉。
- 不使用与参考图冲突的明亮单色主题。
- 不把游戏主画面拆成普通后台系统布局。

## 当前实现对应

- Canvas 主画面负责赛道、车辆、路肩、草地和小地图。
- React HUD 负责左侧排名、右上计时、右侧分段、右下仪表和操作按钮。
- Vue 控制台作为硬件调试面板，必须保持黑色转播信息块风格。
